# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful in issues, pull requests, and discussions. Assume good intent, and when in doubt, ask.

## Enforcement

Report unacceptable behavior to the project maintainer at **loss.of.loss@gmail.com**. All reports are reviewed and handled confidentially. The full enforcement guidelines are on the Contributor Covenant site linked above.
